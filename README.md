# Selenium UI Automation Tests – Swag Labs

This project contains automated UI tests for the Swag Labs demo application using Selenium WebDriver, C#, and NUnit.

## Project Overview

The automation suite validates core user flows in the Swag Labs application including authentication, product management, and cart functionality.

## Automated Test Scenarios

The following test scenarios are covered:

- Login with valid credentials
- Login with invalid credentials
- Login with locked user
- Logout functionality
- Add product to cart
- Add multiple products to cart
- Remove product from cart

## Tech Stack

- C#
- Selenium WebDriver
- NUnit
- ChromeDriver
- Visual Studio

## Project Structure

The project contains the following main components:

- **Tests.cs** – contains all automated UI tests
- **Setup()** – initializes the Chrome browser and navigates to the application
- **TearDown()** – closes the browser after each test execution

## How to Run the Tests

1. Clone the repository
2. Open the project in Visual Studio
3. Restore NuGet packages
4. Run the tests using **Test Explorer**
